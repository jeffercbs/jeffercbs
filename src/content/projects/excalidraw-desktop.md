## Descripción del Proyecto

**Excalidraw Desktop** es una adaptación de [Excalidraw](https://excalidraw.com), la popular herramienta de diagramación y pizarra virtual de código abierto, empaquetada como aplicación nativa de escritorio usando **Tauri**.

El objetivo fue llevar la experiencia de Excalidraw fuera del navegador, permitiendo usarla como una app nativa en Windows, Linux y macOS, con acceso directo desde el sistema operativo sin depender de una pestaña del navegador.

## Tecnologías

- **Tauri** — framework para construir aplicaciones de escritorio con tecnologías web y un backend en Rust, resultando en binarios ligeros y de alto rendimiento
- **Excalidraw** — librería de pizarra virtual con soporte para diagramas de estilo dibujado a mano, colaboración en tiempo real y exportación de archivos

## Características

- Funciona en **Windows, Linux y macOS**
- Interfaz idéntica a la versión web de Excalidraw
- Acceso nativo desde el sistema operativo
- Binario ligero gracias a Tauri y Rust
