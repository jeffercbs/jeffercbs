## Descripción del Proyecto

**Ar Bella** es una tienda en línea de maquillaje construida con una arquitectura desacoplada: frontend en **Next.js** y backend en **NestJS**.

El proyecto cubre el flujo completo de un e-commerce, desde la navegación del catálogo hasta la gestión de pedidos, con una interfaz moderna y optimizada para conversión.

## Tecnologías

- **Next.js** — frontend con renderizado híbrido (SSR/SSG) para SEO y rendimiento
- **NestJS** — API REST en el backend con arquitectura modular y tipado fuerte con TypeScript
- **PostgreSQL** — base de datos relacional para productos, usuarios y pedidos
- **TailwindCSS** — estilos utilitarios para una UI consistente y responsiva
- **JWT** — autenticación segura con tokens

## Características

- Catálogo de productos con filtros por categoría
- Carrito de compras y gestión de pedidos
- Panel de administración para gestión de inventario
- Autenticación de usuarios
- Diseño responsivo optimizado para móvil
